# aionic-playground

Provisioning playground for Ansible / bIONIC beaver.

## Prequirements

* [Ansible](https://www.ansible.com/)
* [Vagrant](https://www.vagrantup.com/)
* [VirtualBox](https://www.virtualbox.org/)

## Fire up VMs

```bash
λ vagrant up
```

## Provision

```bash
λ ansible-playbook provision.yaml
```
